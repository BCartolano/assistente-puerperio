# TASKS - PROJETO CHATBOT MÉDICO

## [PM]

Crie a visão do produto, objetivo, público-alvo e o PRD completo do chatbot médico de atendimento automático com foco em identificação de especialidade, agendamento de consultas, integração WhatsApp e sistemas internos do consultório.

## [PO]

Crie o backlog completo com épicos, histórias de usuário e prioridades para o sistema de chatbot médico, organizando as funcionalidades de identificação de especialidade, agendamento, integração WhatsApp e sincronização com sistemas externos.

## [ANALYST]

Defina todas as regras de negócio, fluxos, validações e exceções para o sistema de agendamento médico via WhatsApp, incluindo identificação de especialidade, validação de horários comerciais, sincronização com sistemas externos e tratamento de casos de emergência.

## [ARCHITECT]

Crie a arquitetura completa do sistema com:
- Backend (Flask/Python)
- Frontend (Web responsivo)
- Banco de dados (SQLite com migração para PostgreSQL)
- Integração com WhatsApp Business API
- Integração com sistemas de gestão do consultório
- Sistema de identificação de especialidade médica usando IA

## [UX-EXPERT]

Crie o fluxo de telas, jornada do usuário e experiência do chatbot médico via WhatsApp, incluindo personas, fluxos de conversação, design de mensagens, tratamento de erros e acessibilidade.

## [DEV]

Crie:
- Estrutura da API REST completa
- Endpoints para agendamento, identificação de especialidade e integração WhatsApp
- Banco de dados com modelos de dados (Appointments, ConversationHistory, SpecialtyMapping, etc.)
- Autenticação e autorização
- Integrações com WhatsApp API e sistemas externos
- Serviços modulares (AppointmentService, SpecialtyService, etc.)

## [QA]

Crie todos os cenários de teste, testes de erro e testes de segurança para o sistema completo, incluindo testes unitários, integração, E2E, performance, segurança e compliance LGPD.

## [SM]

Crie o plano de sprints, organização de tarefas e cronograma para desenvolvimento do chatbot médico, priorizando funcionalidades críticas e organizando dependências entre épicos.

## [BMAD-MASTER]

Revise tudo e aponte inconsistências, melhorias e riscos no projeto do chatbot médico, validando alinhamento entre PRD, arquitetura, UX, implementação e testes.

## [BMAD-ORCHESTRATOR]

Organize a execução completa do projeto em etapas automáticas, coordenando todos os agentes para entregar o sistema de chatbot médico de forma integrada e eficiente.
