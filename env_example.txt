# 2. Gere Senha de App: https://myaccount.google.com/apppasswords
#    - Selecione "Mail" e "Outro (nome personalizado)" → "Sophia Chatbot"

# 3. Use essa senha no MAIL_PASSWORD (não a senha normal!)
# 4. O MAIL_USERNAME deve ser seu email Gmail completo
# 5. O MAIL_DEFAULT_SENDER será automaticamente o mesmo que MAIL_USERNAME para Gmail
